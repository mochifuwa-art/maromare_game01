# APTITUDE — v0.1

「得意なことほど、コストがかからない」をコアとした、ローグライク／デッキビルダー。

## 遊ぶ

ローカルで HTTP サーバーを立てて `index.html` を開いてください。

```bash
python3 -m http.server 8000
# → http://localhost:8000/
```

または、`index.html` をブラウザで直接開くだけでもプレイできます。

## v0.1 で実装した機能

- **適性システム**: 4属性（火・水・風・土）×0〜100の適性値で、カードのコストと威力が変動
  - `実コスト = baseCost × (100 / 適性)`
  - `実威力 = basePower × (適性 / 100)`
- **覚醒システム**: 不得意属性のカードを使うほど苦労値が蓄積、100で覚醒
  - 覚醒中は該当属性のコスト 0MP、威力 ×3
  - 1枚使用で覚醒終了・苦労値リセット
- **弱点・耐性**: 弱点 ×2.5 / 耐性 ×0.25
- **2属性複合カード**: 一方でコスト、もう一方で威力が決まる（必ずトレードオフ）
- **覚醒専用カード**: 該当属性が覚醒中のときだけ手札に出現（ショップで入手）
- **集中**: 次の攻撃の基礎威力 +20
- **ラン構造**: 雑魚3戦 → カード報酬 → 雑魚3戦 → ショップ → ボス
